# Keyboard Shortcuts Reference

## Tab Navigation (View Menu)

| Shortcut | Action |
|----------|--------|
| ⌘1 | Servers tab |
| ⌘2 | Gateway tab |
| ⌘3 | System Logs tab |
| ⌘4 | Config tab |
| ⌘5 | Secrets tab |
| ⌘6 | Setup tab |
| ⌘7 | About tab |
| ⌘R | Refresh server status |

## File Menu

| Shortcut | Action |
|----------|--------|
| ⌘⇧C | Reveal config in Finder |
| ⌘⇧L | Reveal logs in Finder |
| ⌘E | Export logs |

## Server Menu

| Shortcut | Action |
|----------|--------|
| ⌘⇧S | Start all servers |
| ⌘⇧X | Stop all servers |
